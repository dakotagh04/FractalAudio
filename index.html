Cómo Funciona

El Fractal

Este proyecto genera un árbol fractal recursivo que se dibuja desde la base hacia arriba, dividiéndose en múltiples ramas en cada nivel.

Reactividad al Audio

El micrófono captura sonido en tiempo real y analiza su amplitud y frecuencias:

  • Amplitud → Controla el nivel de recursión (profundidad del árbol)
  • Frecuencias bajas → Controlan el ángulo de las ramas
  • Frecuencias medias → Controlan la longitud de las ramas
  • Frecuencias altas → Controlan el color (hue)

Tecnologías Utilizadas

• p5.js para gráficos y animación

• p5.sound.js para análisis de audio

• Algoritmo recursivo para generar el fractal

• FFT (Transformada Rápida de Fourier) para análisis de frecuencias
